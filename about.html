About Syntax Coffee ☕

Syntax Coffee is a web project created by Information Systems students for the Web Application Development course. We chose the name “Syntax Coffee” because programming requires correct syntax and precision, just like our coffee is prepared with quality and care. The website allows customers to explore the menu and pre-order easily online.
